安装与配置 #

SAP Quick Launcher 是一款 Windows 桌面应用，安装过程简洁快速。首次使用需要配置 SAP 快捷方式路径。

1
下载安装包

从官网下载 Windows 安装包（支持 MSI 和 EXE 两种格式），双击运行安装程序

2
配置 SAP 路径

首次启动后，点击右上角 ⚙️ 设置按钮 → 选择「配置 SAP 路径」，指定 sapshcut.exe 的完整路径。通常位于 SAP GUI 安装目录下

3
开始使用

路径配置成功后，即可新建 SAP 连接并一键启动

💡
sapshcut.exe 路径示例：C:\Program Files (x86)\SAP\FrontEnd\SAPGUI\sapshcut.exe。如果未安装 SAP GUI，请先安装 SAP GUI for Windows。

连接管理 #

管理您的所有 SAP 系统连接，支持添加、编辑、删除、复制连接，所有凭证本地加密存储。

➕
新建连接

点击右上角「新建」按钮，填写连接名称、系统标识、客户端、用户名等信息

✏️
编辑连接

点击卡片上的编辑图标或右键菜单选择「编辑」修改连接信息

📋
复制连接

右键菜单选择「复制连接」，快速创建相似配置的新连接

🗑️
删除连接

点击删除图标或右键菜单选择「删除」，删除前会弹出确认对话框

🔒
数据安全：所有连接信息（包括密码）均存储在本地 JSON 文件中，不会上传到任何服务器。关闭应用时数据自动保存。

启动连接 #

多种方式快速启动 SAP 连接，告别繁琐的手动登录流程。

🎯
点击启动按钮

点击卡片上的环境色渐变启动按钮，按钮颜色对应不同环境（红=生产，琥珀=测试，绿=开发）

🖱️
双击卡片启动

双击卡片任意位置即可启动连接，无需精确点击启动按钮

⚡
批量启动

进入批量模式（点击左下角选择框），选中多个连接后点击「批量启动」

💡
启动进度：点击启动后，按钮会显示进度动画，约 1 秒后 SAP GUI 自动打开并登录。启动过程中不会重复触发。

环境区分 #

每个连接都有一个环境属性（生产/测试/开发），通过颜色、图标和边框快速区分不同环境，避免误操作。

🛡️ 生产 PRD

  • 启动按钮：红色→橙色渐变
  • 卡片左边框：红色 4px 实线
  • 环境图标：盾牌警告 ShieldWarning
  • 用途：正式生产环境，需谨慎操作

🧪 测试 QAS

  • 启动按钮：琥珀色→黄色渐变
  • 卡片左边框：琥珀色 4px 实线
  • 环境图标：烧瓶 Flask
  • 用途：质量保证测试环境

💻 开发 DEV

  • 启动按钮：翠绿色→青色渐变
  • 卡片左边框：绿色 4px 实线
  • 环境图标：代码 Code
  • 用途：开发调试环境

✨ 视觉效果

  • 鼠标悬停时显示玻璃镜片反光效果
  • 环境色左边框在悬停时保持不变
  • 启动按钮悬停时有环境色阴影

分组管理 #

通过分组将连接按项目、客户或用途分类整理，保持列表整洁有序。

📂
新建分组

点击「分组管理」按钮 → 输入分组名称 → 创建

🔄
移动到分组

右键连接 → 选择目标分组，或拖拽卡片到目标分组

📝
重命名/删除

在分组管理中重命名或删除分组，删除后组内连接移至「未分组」

🔽
折叠/展开

点击分组标题可折叠或展开该分组下的连接列表

固定卡片 #

将常用的连接固定在「最近使用」区域，不受使用频次和时间的影响，始终快速可达。

📌
固定连接

右键点击卡片 → 选择「固定」，卡片将显示 📌 已固定标记（青色），并始终排在最近使用区域最前面

🔓
取消固定

右键已固定的卡片 → 选择「取消固定」，卡片恢复为普通的最近使用排序

⚠️
注意：最近使用区域最多显示 5 张卡片。固定卡片优先占位，剩余位置由最近使用的非固定卡片填充。已固定的卡片不再显示「最近」标记。

快捷键 #

全局快捷键让你无需切换窗口即可呼出应用，保持工作流连贯。

⌨️
全局呼出 Ctrl + Shift + Space

随时呼出或隐藏应用窗口，无需在任务栏中查找

🔍
搜索过滤

在搜索栏中输入关键词，按名称、系统 ID、用户名等多维度快速定位连接

🏠
系统托盘

关闭窗口后应用最小化到系统托盘，继续监听全局快捷键

🔒
单实例模式

只允许运行一个实例，重复启动时自动呼出已有窗口

主题与语言 #

支持深色/浅色主题和 8 种主题色，中英文界面一键切换。

🌙 深色 / ☀️ 浅色

  • 点击标题栏的主题切换按钮在深色和浅色间切换
  • 深色主题：暗黑背景 + 柔和卡片
  • 浅色主题：白色卡片 + 清爽界面
  • 环境色左边框在两种主题下均可清晰辨认

🌐 中 / 英文

  • 在设置中切换中文和英文界面
  • 所有界面文字、提示、标签均支持双语
  • 包括「最近」「已固定」等卡片标签
  • 切换后即时生效，无需重启
🎨
8 种主题色

翠绿、青蓝、靛紫、玫红、琥珀、橙色、石板、锌灰，还支持自定义主题色

✨
玻璃反光效果

鼠标悬停卡片时，三层反光效果（主高光、对角线反光条、中心微光）营造玻璃镜片质感

导入与导出 #

通过 JSON 文件导入导出连接配置，方便备份迁移和多设备同步。

📤 导出

  • 点击右上角「导出」按钮
  • 选择保存位置，导出为 JSON 文件
  • 包含所有连接、分组信息
  • 可作为备份或分享给同事

📥 导入

  • 点击右上角「导入」按钮
  • 选择之前导出的 JSON 文件
  • 自动合并或覆盖现有连接
  • 适合换电脑或重装系统后恢复
⚠️
隐私提醒：导出的 JSON 文件包含用户名和密码等敏感信息，请妥善保管，不要发送给不信任的人。

常见问题 #

Q
启动时提示「路径无效」怎么办？

请在设置中重新配置 SAP 路径，确保指向正确的 sapshcut.exe 文件。路径示例：C:\Program Files (x86)\SAP\FrontEnd\SAPGUI\sapshcut.exe

Q
全局快捷键不起作用？

确认应用已启动并在系统托盘运行。部分安全软件可能会拦截全局快捷键，请将本应用加入白名单。

Q
数据存储在哪里？

所有数据存储在本地 JSON 文件中（位于用户 AppData 目录），不会上传到任何服务器。可通过导出功能备份。

Q
如何迁移数据到新电脑？

在原电脑上导出 JSON 文件，在新电脑上安装应用后导入即可，所有连接和分组信息完整保留。

Q
支持 macOS 或 Linux 吗？

目前仅支持 Windows 平台，因为 SAP GUI（sapshcut.exe）仅提供 Windows 版本。

Q
密码安全吗？

密码存储在本地配置文件中。建议通过导出功能定期备份，并妥善保管导出文件。如需更高安全性，可在 SAP 系统端启用 SSO 单点登录。
