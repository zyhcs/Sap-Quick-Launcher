# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-02

### Added
- 一键启动 SAP GUI 连接
- 安全凭证本地加密存储
- 多环境管理（PRD/QAS/DEV）
- 深色/浅色主题切换
- 中文/英文双语支持
- 全局快捷键 `Ctrl+Shift+S`
- 系统托盘支持
- 智能搜索功能

### Technical
- 基于 Tauri v2 + React + TypeScript + TailwindCSS 构建
- 支持 Windows .exe 和 .msi 安装包
